3. Platform Overviews

3.1 Cloudflare Pages

One of the world's largest internet infrastructure companies. Hosts websites for free on 330+ data centers worldwide. Used by Discord, Shopify, DHL.

3.2 Netlify

Popular hosting platform for modern websites. Drag-and-drop upload or connect to GitHub. 150+ CDN locations. Used by Google, Nike, Verizon.

3.3 WordPress.com

Managed CMS hosting with a visual editor. Would require rebuilding our site using WordPress themes. Used by CNN, TechCrunch, BBC America.

3.4 Bluehost

Traditional shared hosting company, officially recommended by WordPress.org. Can host static files via FTP, or rebuild site using WordPress. Powers 2M+ websites.

Key Difference: Cloudflare and Netlify host our existing code as-is (no changes needed). WordPress.com and Bluehost would require rebuilding the entire site from scratch.

Glossary

Static Website Fixed files (HTML, CSS, JS). No database. Like a PDF — same for everyone.
CDN Content Delivery Network — copies of your site on servers worldwide for speed.
SSL / HTTPS The 🔒 padlock in browsers. Encrypts data between visitors and your site.
DNS Internet's phone book. Translates setconnect.ai to a server address.
Bandwidth Data transferred when people visit. More visitors = more bandwidth.
Git / GitHub Code tracking system. Like "Track Changes" in Word, but for code.
Deploy Publishing your website — making changes go live.
Rollback Reverting to a previous version. Like "Undo" for your whole website.
CMS Content Management System for editing without coding (e.g., WordPress).
FTP Old method of uploading files to a server.
Shared Hosting Multiple websites share one server. Cheaper but can be slower.
